Hi, I'm Oli. I travel the world and work on coding projects as a digital nomad.

My Location Independence Journey

  • 2013 - Graduated UC Davis with a degree in Environmental International Relations and Psychology.
    Started first job as a business systems analyst. Had a lot of downtime so started learning HTML/CSS at work out of boredom.
  • 2014 - Built first website following online tutorials. Got a new job at a startup developing white label mobile apps using a combination of scripts and tools.
  • 2015 - Read the Tim Ferris 4hr Work Week while on a family trip to Thailand. Started thinking about how to become location indepenedent while on a long boat trip to Kho Phi Phi Island. I realized that programming would allow me to create the office-free lifestyle I wanted to create for myself. I started hitting the books, hard.
  • 2016 - Got promoted to Production Support Engineer and was sent to Amsterdam to learn about microservices. Completed a Web Development Programming Nanodegree.
  • 2017 - The start of my learning vacation where I started traveling South America and working on coding projects on the road. In Colombia I built a Rails Web App called KnowYourself that helps you write your own autobiography. Then I went on a solo backpacking trip through Peru and built a yoga portfolio website for a girl I met at a yoga hostel.

  • 2018 - Spent first 3 months drinking coconuts on the beach in Brazil. Came back to Colombia where I learned React and built a responsive website for a cannabis compliance startup from California. Now I'm in Argentina drinking delicious Malbec wine and actively looking for remote frontend developer opportunities. The coding my way through South America adventure continues!
